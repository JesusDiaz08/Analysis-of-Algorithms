Algoritmo de Kadane

Programación Dinámica Algoritmos Arreglos
Escribe los números separados por un espacio en blanco. 
                
                    int Kadane(int & A[], int N){
                        int maximo_general = A[0]; 
                        int maximo_pos_actual = A[0];
                        
                        for(int i = 1; i < N; i++){
                            maximo_pos_actual = max(maximo_pos_actual + A[i], A[i]);
                            maximo_general = max(maximo_pos_actual, maximo_general);
                        }
                    }
